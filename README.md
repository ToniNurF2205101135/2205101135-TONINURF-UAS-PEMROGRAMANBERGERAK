**Aplikasi Sewa PS** adalah aplikasi berbasis Android yang digunakan untuk melakukan penyewaan PlayStation (PS) secara digital.  
Aplikasi ini dibuat sebagai bagian dari **Ujian Akhir Semester (UAS)** mata kuliah **Pemrograman Bergerak**.

### Pengguna dapat:
- Melihat informasi jenis PS
- Melakukan penyewaan PS
- Melihat struk pembayaran
- Melihat riwayat penyewaan (history)
- Menghubungi kontak rental PS
